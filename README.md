# 1ª Entrega - PWM de Software

Iniciamos um projeto via SMT32CubeMX e realizamos a implementação de um codigo de PWM executado via software, onde o brilho do led é aumentado e diminuido pelo atributo global `DUTY_CYCLE_PERCENT`.
